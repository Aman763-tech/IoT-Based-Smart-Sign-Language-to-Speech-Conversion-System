# IoT-Based-Smart-Sign-Language-to-Speech-Conversion-System

📌 Overview
The IoT-Based Smart Sign Language to Speech Conversion System is a wearable assistive technology designed to help speech- and hearing-impaired individuals communicate effectively.
This system uses smart gloves equipped with flex sensors to detect hand gestures and convert them into real-time text and speech output.

🎯 Objectives
Enable real-time translation of sign language
Bridge the communication gap for differently-abled individuals
Develop a low-cost, portable, and efficient solution
Integrate IoT for wireless communication

🚀 Features
✋ Real-time hand gesture recognition
🔊 Gesture-to-text and speech conversion
📡 Wireless communication via Bluetooth
🧤 Lightweight and wearable glove design
⚡ Fast and responsive system

⚙️ Working Principle
Flex sensors detect finger bending and movement.
The microcontroller (Arduino/ESP32) reads analog sensor values.
Data is processed and compared with predefined gesture patterns.
Matching gestures are converted into corresponding text.
Output is sent via Bluetooth to a mobile device.
Text is converted into speech using a mobile or speaker module.
